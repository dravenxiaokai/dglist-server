# MEAN全栈开发

## 商品列表和点赞功能开发

### ** 技术栈 **
<ol>
    <li>node/express 开发后台api</li>
    <li>mongoose 获取操作mongodb数据</li>
    <li>angular cli 搭建前台框架</li>
    <li>proxy 解决跨域请求，前后端分离，angular路由与nodejs获取请求两不误</li>
</ol>

## 生产环境启动服务
1. Run `node ./src/app.js` 
2. Navigate to `http://localhost:3000/`.
